## 📦 Flagship Project Build Order

</div>

1. **Breast Cancer Logistic Regression (July)**
   - *Focus:* Logistic Regression fitting, StandardScaler pipeline, predict vs predict_proba, metric diagnostics, ROC/PR curves.
2. **Titanic Survival Prediction (August)**
   - *Focus:* Missing value imputation, encodings, ColumnTransformers, pipeline debugging, hyperparameter search, and Streamlit model deployment.
3. **Credit Card Fraud Detection (September)**
   - *Focus:* Extreme class imbalance handling, SMOTE, class weighting, threshold moving, XGBoost/LightGBM, and SHAP explainability.
4. **Employee Churn Prediction (September)**
   - *Focus:* End-to-end regression & classification pipelines, model ensemble, stacking, and feature importance interpretations.
5. **Customer Segmentation (October)**
   - *Focus:* K-Means clustering, PCA visualization, silhouette scores, cluster business profiling, and segmentation reports.
6. **MNIST Digit Classifier (November)**
