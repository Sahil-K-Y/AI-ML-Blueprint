## 🎯 What Is This?

The **AI/ML Blueprint** is a **structured, self-paced 203-day roadmap** designed to take you from absolute zero to a job-ready AI/ML Engineer. Each day has a focused notebook covering theory + hands-on practice.

> 📌 **Goal:** Build deep understanding of AI/ML concepts through consistent daily practice with real datasets and projects.

---

## 🗺️ Roadmap Overview

<div align="center">

```
203 Days Journey
════════════════════════════════════════════════════════════
Phase 1  [Days 001-030] → Python for ML + EDA + Data Wrangling
Phase 2  [Days 031-080] → Classical ML Algorithms
Phase 3  [Days 081-110] → Ensemble Methods + Feature Engineering
Phase 4  [Days 111-150] → Deep Learning (CNNs, RNNs, LSTMs)
Phase 5  [Days 151-180] → Transformers + NLP + LLMs
Phase 6  [Days 181-203] → MLOps + Deployment + Real Projects
════════════════════════════════════════════════════════════
```

</div>

## 📚 Detailed Phase Breakdown

### 🔵 Phase 1 — Python & Data Foundations `Days 001–030`

| Day Range | Topics |
|:---:|:---|
| 001–010 | Python for Data Science — NumPy, Pandas, Matplotlib |
| 011–020 | Exploratory Data Analysis (EDA) — Statistics, Visualization |
| 021–030 | Data Wrangling — Cleaning, Preprocessing, Feature Scaling |

---

### 🟢 Phase 2 — Classical Machine Learning `Days 031–080`

| Day Range | Topics |
|:---:|:---|
| 031–040 | Linear & Logistic Regression |
| 041–050 | Decision Trees & K-Nearest Neighbors |
| 051–060 | Support Vector Machines (SVM) |
| 061–070 | Clustering — K-Means, DBSCAN, Hierarchical |
| 071–080 | Model Evaluation — Cross-Validation, Metrics, Pipelines |

---

### 🟡 Phase 3 — Ensemble Methods `Days 081–110`

| Day Range | Topics |
|:---:|:---|
| 081–090 | Random Forests & Bagging |
| 091–100 | Gradient Boosting — XGBoost, LightGBM, CatBoost |
| 101–110 | Feature Engineering & Dimensionality Reduction (PCA, LDA) |

---

### 🔴 Phase 4 — Deep Learning `Days 111–150`

| Day Range | Topics |
|:---:|:---|
| 111–120 | Neural Networks Fundamentals — Backprop, Activation Functions |
| 121–130 | CNNs — Image Classification, Transfer Learning |
| 131–140 | RNNs & LSTMs — Sequence Modeling |
| 141–150 | Autoencoders & GANs |

---

### 🟣 Phase 5 — Transformers & NLP `Days 151–180`

| Day Range | Topics |
|:---:|:---|
| 151–160 | Attention Mechanisms & Transformer Architecture |
| 161–170 | BERT, GPT — Fine-tuning Pre-trained Models |
| 171–180 | NLP Tasks — Text Classification, NER, QA |

---

### ⚙️ Phase 6 — MLOps & Deployment `Days 181–203`

| Day Range | Topics |
|:---:|:---|
| 181–190 | MLflow, Model Versioning, Experiment Tracking |
| 191–197 | Docker, FastAPI, REST API Deployment |
| 198–203 | Capstone Projects — End-to-End ML Pipeline |
